# parser


## Installation

Запуск базы, билд приложения
```bash 
make docker-up
```

Билд приложения
```bash 
make build
```

Запуск приложения 
```bash 
make run-local
```

Запуск миграций
```bash 
brew install dbmate - установка dbmate
```

```bash 
make migrate
```

## Commands

Откат миграций
```bash 
make migrate-down
```

Остановка приложения базы
```bash 
make docker-down
```
